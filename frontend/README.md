# Retail Sales Management System - Frontend

## Overview
React-based frontend for the Retail Sales Management System with advanced search, filtering, sorting, and pagination capabilities.

## Tech Stack
- React 18
- Vite
- Axios
- CSS3

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file (optional):
```
VITE_API_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

The application will run on `http://localhost:3000`

## Features
- Real-time search across customer names and phone numbers
- Multi-select filters for regions, categories, tags, and more
- Age range and date range filtering
- Sorting by date, quantity, and customer name
- Pagination with 10 items per page
- Responsive design
